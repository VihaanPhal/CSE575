# Data Directory

Place `movie_titles.csv` in this directory.

## CSV Format

```
MovieID,YearOfRelease,Title
```

- No column headers in the file
- `MovieID`: integer
- `YearOfRelease`: integer or `NULL`
- `Title`: string (movie name)

Example rows:
```
1,2003,Dinosaur Planet
2,2004,Isle of Man TT 2004 Review
3,1997,Character
```
