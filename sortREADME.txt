IDrec.py is the user based recommendation file, so you input a userID and get recommendations

cse575_sorter.py is the file which takes the movielens database and sorts it into the format to be used by IDrec.py

movielens_combined.csv is the product of cse575_sorter.py and used by IDrec.py

all other files are taken directly from movielens.

NOTE* the other README is for the links.csv, movies.csv, ratings.csv, and tags.csv files from movie lens not for our project, but I left it there for the sake of completeness.
